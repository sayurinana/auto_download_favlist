# Python 项目规范守则套件

本目录提供五类Python项目的规范守则模板，用于指导AI代理或开发团队快速落地一致的工程实践。文档结构参考`criterion-rust`体系并结合Python生态特性，在每个方向中提供原则、推荐工具、测试与交付策略以及参考资料。

## 内容总览
- `规范守则_CLI与自动化脚本.md`
- `规范守则_Web与微服务.md`
- `规范守则_数据科学与机器学习.md`
- `规范守则_企业级平台与服务.md`
- `规范守则_高性能与分布式计算.md`

## 使用建议
1. 在启动新项目或改造既有项目时，先根据目标类型选择对应守则。
2. 将“核心原则”作为设计评审与代码审查的检查清单，必要时补充团队特定约定。
3. 结合“推荐工具/框架”与“测试与交付”章节，建立自动化流水线与质量保障。
4. 所有引用的在线资源均在每篇守则末尾列出，可持续更新。
5. 若新增方向或调整既有守则，请同步更新本README与相关索引。
