# 现在的任务

编写一个爬虫程序，使用python，

功能是传入一个链接，这个链接是一个Bilibili的用户收藏夹，

比如：https://space.bilibili.com/234561771/favlist?fid=3670113371

这次开发过程就使用上面这个链接作为测试，这是一个有效的Bilibili的用户收藏夹地址，

传入链接后，需要加载收藏夹网页，获取此收藏夹名，收藏夹内视频数，

然后开始获取所有的收藏夹内视频的BV号或AV号（如果有的话），例如对于链接https://www.bilibili.com/video/BV1WD42157Ub/?spm_id_from=333.1387.favlist.content.click，它的BV号就是BV1WD42157Ub，

把获取到的BV号存入一个csv文档中，同时需要记录：
- 获取时的时间戳（使用yyyy-MM-ddThh-mm-ss格式）
- 视频BV号
- 视频标题
- 所属收藏夹名

已存储的BV号不重复记录，csv文件默认使用GBK编码，支持通过参数指定编码格式

注意:
1. 通常情况下，收藏夹页面一页只显示40个视频，当超过40时，将会分页，但我发现点击不同页码时，虽然会出现对应页码的新视频，但是地址栏的URL没有变化
2. 当前开发环境是ubuntu，但是后续此程序需要部署在Windows平台，开发过程中如果有缺少的程序或文件请自行下载无须询问用户，开发完成后需要编写一份指导文档，指导用户如果在windows平台部署和运行此脚本，
